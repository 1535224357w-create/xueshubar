# 学数 bar Design System

## Color Palette
- Primary: `#1a237e` (deep blue)
- Primary Light: `#283593`
- Primary Dark: `#0c1033`
- Accent: `#ff6f00` (warm orange)
- Accent Light: `#ffa040`
- Background: `#f5f6fa`
- Surface: `#ffffff`
- Text: `#1a1a2e`
- Text Muted: `#6b7280`

## Typography
- Body: Inter, -apple-system, PingFang SC, Microsoft YaHei, sans-serif
- Math Symbols: Times New Roman, serif
- Base size: 16px
- Scale: 0.875rem (small), 0.9rem (body), 1rem (lead), 1.5-2.8rem (display)

## Spacing
- Section padding: 1.5rem
- Card padding: 1.4rem
- Grid gap: 1rem
- Container max-width: 1100px

## Corners
- Cards: 18px
- Buttons/inputs: 8px
- Small elements: 6px

## Shadows
- Card default: `0 1px 2px rgba(0,0,0,0.02)`
- Card hover: `0 8px 24px rgba(0,0,0,0.05)`

## Icons
- Math/Unicode symbols only (∫ ∑ ∞ π ∂ ∆ ⊦ ⊢ ⊚ ⊞)
- No emoji as functional icons
